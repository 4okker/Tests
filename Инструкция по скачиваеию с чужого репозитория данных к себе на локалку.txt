Как сделать pull request без fork’а репозитория?
 Pull request это отправка ваших изменений, но без вливания кода сразу в репозиторий. Вначале ваш commit проверят разработчики.

 $ git commit -m "dev 2.0"
On branch master
Changes not staged for commit:
        modified:   C++test.txt

$ git add .

$ git commit -m "dev 2.0"
$ git push m_4test HEAD:Dev
На github’е на странице репозитория выбрать вашу ветку и нажать кнопку «Pull request» (вверху на панели управления).
Уточняя комментарием, что именно Вы изменили и кликаете на кнопку «Send request».
